# ☕ Exercícios de Herança e Polimorfismo em Java

Este repositório contém resoluções de exercícios práticos focados nos pilares da **Programação Orientada a Objetos (POO)**, especificamente em Herança, Sobrescrita de Métodos e Encapsulamento.

## 🚀 Tecnologias Utilizadas
* **Linguagem:** Java
* **Ferramentas:** Git, GitHub Desktop
* **Conceitos:** Herança (`extends`), Polimorfismo (`@Override`), Encapsulamento (`private`, `getters/setters`).

---

## 📂 Estrutura do Projeto

O repositório está dividido em duas pastas principais:

### 1. 🎫 Sistema de Ingressos
Simulação de um sistema de vendas de ingressos com diferentes categorias de preços e localizações.
* **Ingresso:** Classe base com valor.
* **VIP / Normal:** Especializações do ingresso base.
* **Camarote (Inferior/Superior):** Subclasses de VIP com lógica de adicionais e localização.

### 2. 🏠 Sistema de Imobiliária
Gestão de imóveis com regras de precificação variadas de acordo com o estado do imóvel.
* **Imovel:** Classe base com endereço e preço de venda.
* **Novo:** Aplica um valor adicional por ser uma construção moderna.
* **Velho:** Aplica um desconto proporcional ao desgaste do imóvel.
